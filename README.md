# Meme_Data_Project
## 모델 구축 방향
* 사용자가 텍스트를 입력하면 텍스트에서 키워드를 추출하여 그 키워드에 적합한 이미지 및 영상등의 매개체를 출력하려고 함.
* 그러면 난 입력 문자에 대한 자연어처리 모델을 구축하여 추출 된 키워드를 동작 코드로 넘겨줘야함.
* 짧은 문장의 키워드 추출이 과연 가능할지는 모르겠지만 최선을 다해봄.
